# Dashboard - Loja de Jogos

Uma dashboard desktop desenvolvida para fins de teste técnico utilizando Tauri + React, com foco em autenticação, organização de componentes e experiência moderna de interface.

---

## Tecnologias Utilizadas

- ReactJS
- Rust
- Tauri
- Node.js
- Git

---

## Como Rodar o Projeto

```bash
git clone https://github.com/WebersonRodrigues7/Projeto-AbtAtividades.git

cd Projeto-AbtAtividades

npm install

npm run tauri dev
```

---

## Funcionalidades

- Dashboard de jogos
- Aplicação desktop com Tauri
- Interface responsiva
- Estrutura organizada e componentizada

---

## Objetivo do Projeto

Este projeto foi desenvolvido como parte de um teste técnico, com o objetivo de demonstrar conhecimentos em:

- Estruturação de aplicações React
- Integração com Tauri
- Componentização
- Gerenciamento de estado
- Organização de projeto
- Desenvolvimento desktop moderno
